# 🖼️ Creative Image Gallery

A modern and responsive image gallery built using HTML, CSS, and JavaScript. This project provides an interactive gallery experience with category filters, image search, hover animations, lightbox preview, and a visually appealing user interface.

## 🚀 Features

* Responsive Design for Mobile, Tablet, and Desktop
* Modern Glassmorphism UI
* Animated Gradient Background
* Category-Based Image Filtering
* Search Functionality
* Hover Zoom Effects
* Lightbox Image Preview
* Smooth Animations
* Scroll-to-Top Button
* Professional Layout and Design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## ✨ Gallery Functionalities

### 🔍 Search Images

Users can search images by their titles.

### 🏷️ Category Filters

* All
* Nature
* City
* Travel

### 🖼️ Lightbox Preview

Clicking on an image opens it in a larger preview mode.

### 📱 Responsive Design

The gallery automatically adapts to different screen sizes.

### ⬆️ Scroll to Top

Quick navigation back to the top of the page.

## 💻 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

OR

Use the Live Server extension in VS Code.
